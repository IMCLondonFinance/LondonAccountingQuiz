EmailJS Setup

How to set up EmailJS (free, ~5 minutes):

1. Go to emailjs.com and create a free account.
2. Add an Email Service (connect your Gmail, Outlook, etc.) — note the Service ID.
3. Create an Email Template. Use these variables in your template:
  • {{to_email}} — candidate's email
  • {{to_name}} — candidate's name
  • {{admin_email}} — your email (receives a copy)
  • {{score}} — e.g. 19/25
  • {{time_used}} — time taken
  • {{completed_at}} — date/time completed
  Note the Template ID.
4. Go to Account → API Keys and copy your Public Key.
5. Paste all three values below and click Save.
